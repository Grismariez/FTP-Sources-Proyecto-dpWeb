Nuestra página tiene una estructura de Tipo Red:

                                  


Una página web tiene elementos estructurales estándar que dan información básica sobre el sitio web, su operador y el contenido. También podría mostrar anuncios, contenido agregado o vínculos a otro material que sea igualmente relevante para los visitantes del sitio. Las estructuras principales que cumplen estas funciones tienen formas, tamaños y lugares que reflejan su importancia en la página web en particular, mientras que otras reflejan la estructura de la información del sitio web en general.
Básicamente existen tres modelos de estructura que podemos utilizar para crear una página web:
  • Modelo Secuencial
  • Modelo Jerárquico
  • Modelo de Red

El Modelo Secuencial:

Es la forma más fácil de catalogar toda la información que queremos ofrecer a los usuarios.
El contenido se ordena cronológicamente, es decir, por fechas, dejando paso a que la información actualizada recientemente aparezca al principio y la más antigua al final. No obstante, también se puede ordenar siguiendo unas directrices lógicas, por ejemplo, siguiendo el alfabeto.

El Modelo Jerárquico:

Se utiliza para coordinar grandes dimensiones de información. Por ello, el tipo de personas que optan por esta estructura de página web son aquellas que tienen mucha información que ofrecer (sitios comerciales o tiendas online) y se destacan por mostrar a los usuarios sus productos o características desde las más genéricas hasta las más específicas.
Una de las ventajas de escoger este tipo de estructura web es que la página índex puede recibir enlaces entrantes del resto de contenidos propios de la web. Por lo tanto, la página de inicio (llamada también: principal, home o índex) se beneficia de todos los enlaces internos que recibe desde otras páginas. El resultado será una mayor popularidad de la «principal» que quedará reflejado en la puntuación del Page Rank.

El Modelo de Red:

Es uno de los más manejables a la hora de administrar la información. Con este tipo de diseño web se pretende incentivar y favorecer la libre navegación sin patrones establecidos. La finalidad de este modelo de estructura web es explotar al máximo los contenidos elaborados.
Aunque es cierto que, éste tipo de estructura es muy útil para mantener a los usuarios en la web durante un período de tiempo más prolongado, también hay que tener en cuenta que puede causar confusión, ya que el orden de la web puede parecer, en primera instancia, más complicado de identificar. Este conflicto puede aumentar si el visitante no toma la iniciativa e interes de ser curioso para familiarizarse con los menús de navegación del sitio; ya que una web bien estructurada posee la ventaja de ser amigable ante esta situación de adaptación del usuario.
La principal diferencia del modelo jerárquico con la estructura en red es que el reparto de Page Rank es más ecuánime. Además, al contrario que los otros modelos de webs anteriores, cualquier página puede catalogarse como entrada o salida, e igualmente, es más fácil saltar de una categoría a otra.
